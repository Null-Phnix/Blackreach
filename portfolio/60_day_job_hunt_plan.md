# 60-Day Job Hunt Plan: Land a Remote AI/Python Role

## Week 1-2: Portfolio Polish + Setup

### Day 1-3: GitHub Cleanup
- [ ] Make Blackreach repo public
- [ ] Write killer README.md (see template below)
- [ ] Add screenshots/demo GIF of it running
- [ ] Clean up commit history if needed
- [ ] Pin it to your profile

### Day 4-5: LinkedIn Profile
- [ ] Update headline: "Python Developer | AI Automation | Browser Agents"
- [ ] Add Blackreach to Projects section with link
- [ ] Post about building it: "Just shipped Blackreach - an autonomous browser agent with 2800+ tests..."
- [ ] Connect with 20 recruiters in Toronto/Remote AI space

### Day 6-7: Resume + Cover Letters
- [ ] Finalize resume using template
- [ ] Save 3 versions: (1) General Python, (2) AI Engineer, (3) QA/Automation
- [ ] Prepare cover letter templates in a doc for fast customization

---

## Week 3-4: Application Blitz

**Daily Goal: 10 applications/day**

### Sites to Hit Every Morning (30 min)
1. **Wellfound** (filter: Canada, Remote, Entry-level, AI/Python)
2. **LinkedIn Easy Apply** (filter: Remote, Entry-level, Last 7 days)
3. **YC Jobs Board** (workatastartup.com)
4. **RemoteOK** (remote-python-jobs, remote-ai-jobs)
5. **Built In** (Toronto/Vancouver tech jobs)

### Application Workflow (3 min per job)
1. Read job description (30 sec)
2. Pick matching cover letter template (30 sec)
3. Customize [COMPANY], [ROLE], one specific detail (90 sec)
4. Submit with tailored resume (30 sec)

**Weekly Target: 50 applications**

---

## Week 5-6: Follow-Ups + Interview Prep

### Follow-Up Strategy
- Day 7 after applying: Send LinkedIn message to recruiter/hiring manager
- Template: "Hi [Name], I applied for [Role] last week. I built a production browser agent (2800+ tests) and would love to discuss how this experience applies to [Company]'s work. Are you open to a brief chat?"

### Interview Prep (Daily 1 hour)
- **Coding practice:** LeetCode Easy (Arrays, Strings, Hash Maps) - 2 problems/day
- **System design:** How would you build [job board scraper, API rate limiter, download manager]
- **Behavioral prep:**
  - "Tell me about Blackreach" (30 sec elevator pitch)
  - "Tell me about a hard bug you fixed" (Phase 1 race conditions from your work)
  - "How do you test your code?" (Point to 2868 tests)

---

## Week 7-8: Close Deals

### When You Get Interviews
**Take-Home Projects:**
- You + Claude Code do it together
- Deliver in <24 hours
- Over-deliver: Add tests, README, edge case handling

**Technical Screens:**
- Be honest about using AI tools: "I use Claude/Copilot for productivity, happy to code with or without"
- Most companies don't care if you're transparent

**Offer Negotiation:**
- First offer: "Thank you! Can I have 48 hours to review?"
- Counter: Ask for +$5-10k or better benefits
- Don't accept first offer immediately

## Backup Plan (If No Offers by Week 6)

### Pivot to Freelance/Contract
- Post on Upwork: "Browser Automation Expert | Python + Playwright"
- Charge $50-75/hour
- Do 2-3 small projects ($500-1500 each)
- Use these as "work experience" on resume
- Re-apply to jobs with "Currently freelancing as Python automation engineer"

### Build One More Demo
- "AI Job Application Assistant" - scrapes jobs, writes cover letters, tracks applications
- Post on Twitter/Reddit: "Built this to automate my job hunt"
- Goes viral → companies reach out to YOU

**2 final rounds** × **50% offer rate** = **1 offer**
